Overview

In this lab, we will create a console application and a simple class and test code

  1. Open Visual Studio Community 2022 and choose Create new Project
  2. In the filters, choose C# and Console
  3. Choose a Console application (note - there are 2 project types. .Net Core has multiple platforms while .Net framework is Windows only. Choose .Net Core projects)
  4. Name your project FirstLab
  5. Change the name of the solution to Labs
  6. Change the location of the project to whatever folder works best
  7. Right-click on the new project in the Solution Explorer and choose Add/Class
  8. Name the class Employee
